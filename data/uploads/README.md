# Uploaded Documents

I use this folder to store source documents before running the ingestion process.

Supported file formats:

- PDF
- DOCX
- PPTX
- IPYNB
- JPG, JPEG, and PNG
- XLSX, XLS, and CSV

Files placed in this folder are excluded from Git tracking because they may contain private, licensed, or sensitive material.

Only this README file is included in the public repository.
